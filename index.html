dota 2 api 介绍

Dota 2 提供了有关比赛记录和玩家数据的API, 以供开发者用来开发相关软件Dotamax和Dotabuff.

准备工作

在使用dota2 api之前,需要先查看使用规定Steam Web API Terms of Use
如果没有Steam账户,请先申请一个Steam账户,然后去http://steamcommunity.com/dev/apikey申请一个开发者使用的apikey.

Api

Dota 2 Api返回的数据格式分为json和xml两种,开发者可以根据自己的情况来选择,这点在后面api参数会有介绍.
